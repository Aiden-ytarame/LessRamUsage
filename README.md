Decreases ram usage by only loading level album and song when needed, otherwise its unloaded. 

With this the game uses a consistent 1.5Gb~ of ram instead of scaling massively with the more levels you have. On my machine with 800~ levels I use 7Gb of ram without the mod, and around 1.4Gb with the mod. 

Tested the mod with 3000 levels and it runs perfectly
